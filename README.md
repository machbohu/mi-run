Solution:
---------

    4. AST interpreter

    Implemented:
    ------------

    - boolean
    - string
    - [] (overflow protection)
    - classes
    - methods
    - I/O
    - Garbage Collector (generational scavenging)
    - ||, &&, !
    - if, else, if-else
    - for, while
    - final
    - static methods
